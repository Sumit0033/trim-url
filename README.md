# URL Shortener App

A web-based URL shortener application developed using Node.js, Express, EJS, and SQLite. This project is designed to run locally and demonstrates the implementation of a complete web application with server-side rendering and database integration.

## 👨‍💻 Author  
Sumit (GitHub: Sumit0033)

---

## 📌 Description

This application allows users to convert long URLs into short, manageable links. Each shortened URL redirects users to the original destination while also tracking how many times the link has been accessed.

The backend is built using Express.js, which handles routing and server-side logic. EJS is used for rendering dynamic web pages, and SQLite is used as a lightweight local database to store URL mappings and click statistics.

The application also includes basic validation and security checks such as rejecting invalid or insecure URLs.

---

## 🚀 Features

- Shorten long URLs into unique links  
- Redirect short links to original URLs  
- Track click counts  
- Delete saved URLs  
- Input validation and basic security  

---

## 💻 Tech Stack

- Node.js  
- Express.js  
- EJS (Embedded JavaScript Templates)  
- SQLite (Local Database)
